**DataSpark: Global Electronics Insights**
**Overview**
This project focuses on analyzing global electronics trends using data cleaning, merging, and visualization techniques. The final deliverables include an Excel file with merged data, SQL analysis, Power BI visualizations, and an EDA report.

**Features**
Data Cleaning: Handling missing and inconsistent data.
Data Merging: Integrating multiple datasets into a single Excel file.
SQL Analysis: Performing SQL queries to extract insights.
Power BI Dashboards: Visualizing data trends interactively.
EDA Report: Summarizing key insights from data analysis.

**Technologies Used**
Python (Pandas, NumPy)
SQL
Power BI
Excel

**Usage**
Run SQL scripts for data analysis.
Open Power BI dashboards for visualizations.
Review the EDA report for detailed insights.
